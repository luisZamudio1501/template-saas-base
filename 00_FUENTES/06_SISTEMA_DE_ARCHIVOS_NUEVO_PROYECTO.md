# 📄 06_SISTEMA_DE_ARCHIVOS_NUEVO_PROYECTO.md

## 🎯 OBJETIVO

Definir la estructura obligatoria de carpetas y documentos para todo nuevo proyecto creado desde el template base.

Este documento asegura:

- orden en la fábrica
- reutilización real
- continuidad entre proyectos
- separación entre sistema y producto

---

## 🧱 PRINCIPIO CLAVE

Todo proyecto se divide en:

- 00_FUENTES → reglas del sistema
- 01_DOCUMENTOS → construcción del producto

---
