# 00_FABRICA_DE_APPS

## Descripción
Sistema base para la creación de aplicaciones SaaS.  
Este proyecto define la estructura, arquitectura y componentes necesarios para construir productos digitales escalables.

## Objetivo
Construir una fábrica de SaaS que permita:
- Crear aplicaciones rápidamente
- Reutilizar estructuras base
- Escalar con backend desacoplado (Next.js + servicios + motor + Spring Boot)
- Integrar automatización (n8n) e inteligencia artificial

## Estructura del proyecto

- template-base-v1/
  - Base técnica inicial del sistema
- Documentos (.docx)
  - Definición estratégica
  - Diseño de agentes
  - Plantillas de invocación

## Tecnologías consideradas

- Frontend: Next.js
- Backend: Spring Boot (fase avanzada)
- Base de datos: Supabase / SQL
- Automatización: n8n
- IA: integración con APIs
